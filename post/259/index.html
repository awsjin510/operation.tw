AI12｜簡報製作第三世代：用 GPT-Image-2 把每張投影片畫出來

2026-05-01

不是用 ChatGPT 產 PPT 檔，也不是在 Gamma 套模板——而是把每一張投影片當作一幅圖像直接生成。這集「操作一下」聊的，是一個可能讓你回不去的工作流：用 GPT-Image-2 一次生成十張風格一致的投影片，品質比花半天套 Canva 模板還要穩。五個步驟、雷區提醒、三種身分的應用場景，完整拆解。

你上一次做完簡報之後，有沒有覺得哪裡不對勁？

不是內容不好，而是視覺——字體選了又換、配色試了又改、圖片授權不確定、模板套上去又不像自己的風格。簡報的「做完」和「做好看」之間，常常隔著一個下午的時間。

這集 Podcast 想聊的，就是這個缺口有沒有新的解法。而答案，來自一個你可能還沒想到的用法：用 GPT-Image-2 把每一張投影片當圖片畫出來。

第三世代簡報是什麼意思？

如果要粗略區分簡報工具的世代，大概是這樣：第一代是 PowerPoint、Keynote，自己拖拉排版；第二代是 Canva、Gamma 這類模板工具，降低設計門檻；第三代，則是把簡報的每一頁視為一張獨立的視覺創作，由 AI 直接生成影像。

這不是請 ChatGPT 輸出一份 .pptx 檔，也不是在既有平台上套版——而是給定內容與視覺風格，讓 GPT-Image-2 把整頁的構圖、排版、色調、插圖一次到位地畫出來。一次十張，風格天生一致，不需要你去「對齊」任何東西。

完整工作流：五個步驟

這集把整個流程拆成五個步驟，讓你明天就能開始用：

  • 寫大綱：先把簡報的每一頁要傳達的核心訊息條列出來，這是後續所有 prompt 的基礎。
  • 定義視覺風格：在下 prompt 之前，先確立整份簡報的基調——是科技感、溫暖插畫、還是極簡商務？這個描述要在每一頁的 prompt 裡保持一致。
  • 下 prompt 生成：每頁各自寫一段 prompt，包含頁面文字內容、視覺構圖方向，以及統一的風格描述。GPT-Image-2 單次有張數限制，需要分批處理。
  • 維持風格一致性：分批生成的挑戰在於每批之間可能出現風格漂移。解法是把第一批生成結果中最滿意的一張當作「視覺錨點」，在後續 prompt 中引用並描述它的特徵。
  • 輸出與後製：生成的圖片可以直接匯入簡報工具作為全頁背景圖，或視需要做輕量後製。

哪些頁面千萬不要這樣做？

這集也很誠實地點出雷區。如果你的投影片需要呈現精確的數據圖表、複雜的表格、或是條列式的技術規格，用圖像生成的方式反而會出問題——數字排列、表格結構、文字精確度都不是 GPT-Image-2 的強項，這類頁面還是回到傳統工具處理比較穩。

換句話說，這個方法最適合的是：封面頁、章節過渡頁、視覺說明頁、氛圍渲染頁——那些你過去最容易花時間找圖、卻又找不到滿意結果的頁面。

它跟 NotebookLM 的差異在哪？

這集也做了一個有趣的比較。NotebookLM 擅長的是從大量文件中萃取與整理內容，是「資料→理解」的流程；而 GPT-Image-2 的簡報工作流，是「內容→視覺表達」的流程。兩者不是競爭關係，而是可以接力——先用 NotebookLM 整理出簡報的骨架，再用 GPT-Image-2 把它畫出來。

對不同身分的人，這條路徑代表什麼？

這集最後針對三種人做了分析。對創作者來說，這是一個不需要設計背景就能產出高品質視覺內容的新入口；對上班族來說，是把「做簡報」從半天的苦差事壓縮成一個下午的可能；對設計師來說，則值得認真思考：這個工具不會取代你的判斷力，但它確實改變了起點——你的價值會更集中在策略與審美決策，而不是執行本身。
