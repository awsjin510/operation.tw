把雲端學習變成一場任務挑戰：我的 AWS Cloud Quest 實戰體驗

2026-01-31

Lab的過程中可以照著圖片步驟執行，而最後的DIY環節則必須自己思考去完成。 不只是複習基礎，更是用遊戲重新校正自己對雲端架構與最佳實務的理解，也很適合想準備 Cloud Practitioner 證照或剛入門雲端的朋友，相信做完一輪會更有概念!!

完成AWS Skill Builder 上的 Cloud Quest: Cloud Practitioner 任務，順利拿到這枚雲端學習徽章~

這次透過遊戲化方式練習Lab，從扮演 Cloud Practitioner 角色、在 3D 虛擬城市裡解決任務，到進入 Solution Center 做實際 Lab，讓原本抽象的服務像 EC2、S3、VPC、IAM、與基本架構概念都「玩」過一輪，更有記憶點。
​

蠻喜歡這個把流程拆成: 了解、練習、DIY的過程

先用簡短內容建立概念(有影片)，再透過hands‑on lab在真實 AWS 環境裡動手操作，最後自己組出完整解決方案，等於一次把理論、實作和情境思維串在一起。
另外覺得不錯的點是Lab的過程中可以照著圖片步驟執行，而最後的DIY環節則必須自己思考去完成。

不只是複習基礎，更是用遊戲重新校正自己對雲端架構與最佳實務的理解，也很適合想準備 Cloud Practitioner 證照或剛入門雲端的朋友，相信做完一輪會更有概念!!
