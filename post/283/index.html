AI33｜《Rewire 神經可塑性》：大腦靠重複動作刻迴路，不是靠想

2026-06-17

你能快速 debug 別人的爛 code，卻對自己三十幾年的壞習慣束手無策。這集介紹英國神經科學家 Nicole Vignola 的《Rewire 神經可塑性》，從神經科學角度解釋為什麼「想改變」不夠用、負面自我對話其實是演化預設值，以及多巴胺分泌的不是快樂而是「渴望」——讀完讓人重新思考改變自己的正確層級在哪裡。

先問你一個問題。

假設你今天剛幫一個跑了五年的老系統做完 refactor。你看得出哪個函式效能最差、哪段邏輯該抽出去、哪個迴圈每次都在重複計算一樣的東西。debug 別人的爛 code，你又快又狠。

但回到家，面對自己這個跑了三十幾年的「老系統」，你卻完全沒辦法動它。明知道不該熬夜、不該滑手機到凌晨兩點、早就該開始運動了——還是日復一日做著一模一樣的事。

為什麼？

這集 Podcast 介紹的書《Rewire 神經可塑性》，作者是英國神經科學家 Nicole Vignola，給了我一個很直接的答案：不是你意志力不夠，是你動到的層級錯了。

大腦不是被「想」就能改變的

很多人以為，只要認知到某個習慣不好、下定決心改，就能改變。但這本書把這件事從心理學的「你要更努力」，拉回到神經科學的「你要動到硬體層」。

大腦改變的機制叫做神經可塑性（neuroplasticity）——神經元之間的連結，會因為反覆被觸發而變得更強、更自動化。用工程語言來說，你每一次重複某個行為，就是在強化那條迴路的權重。光是「想改變」，不會動到任何一條已經被強化的迴路。真正能改變的，是重複不同的行為，讓新的迴路慢慢被刻進去。

這就是為什麼「從明天開始」幾乎不會成功——你需要的不是一個決心，而是夠多次的重複行為，讓新路徑的權重高過舊路徑。

負面內在對話，是大腦的演化預設值

書裡另一個讓我停下來想了很久的觀點，是關於「負面自我對話」。

很多人面對失敗或挫折，第一個反應是自責：我怎麼這麼廢、我就是沒辦法改變。然後又因為這樣責備自己，更難採取行動，形成一個負向循環。

Nicole Vignola 的解釋是：這不是你的問題，是演化的預設值。大腦天生對負面訊號更敏感，這在遠古環境下是生存優勢——對威脅高度警覺，才能活下去。但放到現代生活，這個機制就變成了「對自己的負面濾鏡」。

理解這件事之後，你至少可以先停止把「我又失敗了」解讀成「我這個人有問題」。那只是一個還沒被改寫的預設迴路在運作。

多巴胺分泌的是「渴望」，不是快樂

這是這集裡打到我最深的一個觀點。

我們通常以為多巴胺是「快樂的神經傳導物質」，所以以為刷到有趣的短影片、得到按讚通知，就是在獲得快樂。但事實是，多巴胺的功能是驅動「渴望」——它讓你想要去追求某件事，但追到之後，並不會真的滿足。

這就是為什麼你刷完一個短影片，不會停下來，而是繼續滑下一個。你追的不是滿足感，你追的是「下一個可能更好」的預期。這個機制被演算法設計得非常精準，讓你永遠處於渴望狀態，卻永遠不覺得夠。

知道這件事，不會讓你立刻戒掉短影片。但它至少讓你理解：那種「再看一個就好」的感覺，是神經機制在運作，不是你意志力特別差。

讀完之後，我實際改了什麼

這集最後也分享了讀完這本書之後，我自己調整的三件事。其中我覺得最有用的一個觀念是：設計環境，比靠意志力管用太多。

與其每天要求自己「不要滑手機」，不如讓手機物理上不在你手邊。與其要求自己「要去運動」，不如把運動裝備放在你早上第一眼會看到的地方。行為發生的阻力越低，那條迴路就越容易被觸發。

  • 減少壞習慣的觸發點（讓壞迴路難以啟動）
  • 降低好習慣的執行阻力（讓好迴路容易被走到）
  • 用重複次數累積，而不是靠單次的強烈決心

《Rewire 神經可塑性》這本書讓我覺得最有價值的地方，是它把「改變自己」這件事從道德層次（你要更自律）移到了機制層次（你要用正確的方式動到神經迴路）。當你理解自己在對抗的是什麼，你才有機會設計出真的有效的做法。

這集 Podcast 大約四十分鐘，如果你也覺得自己 debug 別人比 debug 自己容易太多，很推薦聽聽看。
