懶人必看！NotebookLM如何讓AWS證照準備效率暴增

2026-02-19

運用 AI 工具 NotebookLM 加速準備 AWS Certified Machine Learning Engineer – Associate (MLA-C01) 證照

趁著年假的空擋，來準備AWS Certified Machine Learning Engineer – Associate (MLA-C01)證照，近3年期間我考了10張證照，與大家分享如何運用AI透過NotebookLM 如何加速AWS MLA-C01 證照準備。

過往我們在學習一門新知識時，可能會先上網找相關的學習資源，像是：練習題、備考心得、YT影片等，但通常這些資源分散各地，沒有一個可以彙整起來學習或複習的平台，即便我會統一整理在Notion複習，但仍少了許多對應服務的畫面感。

▌那如果我今天想要快速學習一門新知識或技術，有哪些步驟可以參考？
1.掌握脈絡→快速地掌握將要學習的新知識脈。
2.延伸概念→了解這些脈絡的更多延伸概念，清晰有效學習新事物。
3.深挖細節→針對延伸概念深挖探討細節沉澱、吸收。

▌直到Gemini3 推出後，我的學習流程透過 NotebookLM徹底轉型。
我將AWS官方的考試指南、YT備考影片(很棒的一點是英文的就行)、與個人筆記匯入NotebookLM，利用Studio延伸的9個功能，作為智慧學習引擎，自動生成結構化的學習資源，應用情境如下：

一｜『掌握脈絡』
→1.心智圖：架構化拆解考試指南中的單元重點與考試配比。
→2.資訊圖表：圖像化掌握脈絡，增加記憶連結(透過不同的提示詞產出不同版本)

二｜『延伸概念』(透過不同的提示詞產出不同版本)
→3.報告：產生重點複習報告，涵蓋考試總覽、高頻考點、常見的陷阱與誤區、考前知識點Check、必讀官方文件。
→4.簡報：產出AWS備考風格簡報，透過AWS 官方設計語言 + 考點架構 + 視覺層級，讓輸出看起來就像 AWS 內部培訓教材。
→5.語音摘要：用Podcast對話的方式進行重點複習，通勤、運動時可以聽。
→6.影片摘要：依照不同單元產出10分鐘左右的章節介紹、考試重點，圖像搭配聲音的方式學習，增加記憶點。

三｜『深挖細節』(考前總複習)
→7.資料表：產出重點功能與服務表，描述功能用途與場景、特型、優點與考試重點提示。
→8.測驗：貼近實際考試可能出現的情境，模擬考試的狀態。
→9.學習卡：反思學習過程中的知識點、提取記憶。


過程中看到不錯的學習資源，都可以再匯入強化，更可以當作是準備這張證照的知識庫，學一學有不懂的，就直接在對話框詢問，因為都是這張證照的相關資料，所以能夠有效避免AI產生幻覺。

還可以直接把這本筆記本與同事們分享，大家一起透過Notebooklm學習，不用每個人再花時間彙整、專研提示詞。
透過Notebooklm不僅提升效率，也補強知識內化，讓我從「被動閱讀」轉為「互動建構」，更能達到我喜歡與大家分享，培養團隊學習風氣的效果!!
