AI05｜Claude Opus 4.7 四大新功能完整解析：視覺、思考、預算一次搞懂

2026-04-19

Anthropic 最新旗艦模型 Claude Opus 4.7 正式發布，在 13 項主要 benchmark 中拿下 8 項公開模型第一。這篇文章整理了這集 Podcast 拆解的四大核心新功能：三倍解析度視覺、xhigh 思考等級、Task Budgets 預算控制，以及悄悄影響你帳單的新 Tokenizer，幫你快速掌握這次升級的實際意義。

如果你最近有在用 Claude，你可能已經注意到介面上出現了一些新選項。沒錯，Anthropic 悄悄（又不那麼悄悄地）推出了旗艦模型 Claude Opus 4.7，這次的升級幅度相當大，大到值得專門錄一集來拆解。

這次升級有多大？先看成績單

Claude Opus 4.7 在 13 項主要 benchmark 中拿下 8 項公開模型第一。其中最讓人驚訝的是視覺推理能力，一口氣進步了 13 個百分點，程式碼能力也有明顯躍升。這不是那種「比上一版好一點點」的例行更新，而是在幾個關鍵維度上真的拉開了差距。

這集 Podcast 裡，我請出了我的 AI 分身，用它來幫忙拆解這次四大新功能的重點與影響。以下是延伸整理：

功能一：三倍解析度的高畫質視覺

Claude 這次在視覺理解上做了大幅強化，支援更高解析度的圖片輸入，讓模型在分析圖表、截圖、設計稿或文件掃描時，能擷取到更細節的資訊。對於需要讀圖的工作流程來說，這是實質的生產力提升，不再是把一張糊掉的圖丟給 AI 然後得到模糊的答案。

功能二：全新的 xhigh 思考等級

Claude 原本就有「延伸思考」模式，這次新增了 xhigh 這個更高階的思考等級。簡單說，就是讓模型在回答前花更多運算資源「想清楚」，適合用在高複雜度的推理任務、多步驟規劃，或需要嚴謹邏輯的技術問題。當然，更深的思考也意味著更高的成本，這就帶出了第三個功能的重要性。

功能三：Task Budgets——終於可以幫 AI 設預算了

這是這次我覺得最實用的新功能。Task Budgets 讓你可以為單次任務設定思考 token 的上限，也就是說，你可以告訴 Claude「這個任務最多只能用這麼多算力」，避免一個簡單問題卻觸發大量思考、帳單暴增的情況。

對於開發者或是重度使用 API 的用戶來說，這是一個讓成本變得可預期的好工具。你不再需要賭運氣，或是刻意壓低模型能力才能控制費用。

功能四：新 Tokenizer——悄悄讓帳單變貴的那個

這個功能沒有被大肆宣傳，但影響不小。Anthropic 更新了 Tokenizer（文字分詞器），改變了模型把文字切分成 token 的方式。在某些語言或特定格式的文字下，同樣的內容可能會產生更多 token，直接影響你的使用成本。

如果你發現最近帳單有點微妙地變貴，這很可能是原因之一。建議重新測試你常用的 prompt，看看 token 用量有沒有明顯變化。

對你的實際操作意義是什麼？

  • 視覺工作流程：如果你的工作涉及圖表分析、文件讀取或設計審查，現在是重新測試 Claude 視覺能力的好時機。
  • API 使用者：Task Budgets 值得優先研究，搭配 xhigh 思考等級，可以在效果與成本之間找到更精確的平衡點。
  • 所有付費用戶：留意新 Tokenizer 對你常用 prompt 的影響，必要時重新優化提示詞的長度與格式。

總結

Claude Opus 4.7 是一次真正有感的升級，但「有感」這件事是雙向的——能力變強了，成本結構也跟著變了。這集 Podcast 拆解這四個功能的目的，就是希望讓你在升級紅利到來之前，先把坑踩清楚。
